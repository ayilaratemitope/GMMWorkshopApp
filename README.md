## Project Overview
This system was developed as part of a Software Engineering assignment to design and implement a workshop management system using modern web technologies.


# GMM Workshop Management System
A full-stack web application developed using **Blazor Server** and **Entity Framework Core** to manage workshop operations including motorists, vehicles, repairs, and parts.

# Features
- Register and manage motorists
- Register and manage vehicles
- Log and track repairs
- Add and manage spare parts
- Calculate total repair cost (parts only)
- Admin authentication system
- Clean UI with responsive design

# Technologies Used
- ASP.NET Core Blazor Server
- Entity Framework Core
- SQLite / SQL Server
- Bootstrap (UI Styling)
- C#
